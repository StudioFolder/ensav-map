# ENSAV Interactive Map

Interactive visualization of ENSAV's international partnerships, student projects, research collaborations, and mobility programs.

**Client:** ENSAV (École Nationale Supérieure d'Architecture de Versailles)
**Studio:** [Studio Folder](https://studiofolder.it)

## Setup

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:5173`.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Directus as headless CMS (separate repo)
